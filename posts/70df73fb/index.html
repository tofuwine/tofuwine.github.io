使用 Google 搜索时，有许多技巧可以帮助我们更有效地找到所需的信息。

以下是一些常见和高级的 Google 搜索技巧：

基本技巧#

使用引号进行精准匹配#

搜索一个确切的短语："exact phrase"

快速预览：Google 搜索

使用减号排除词#

排除包含特定词的结果：keyword -excludedword

快速预览：Google 搜索

使用星号作为通配符#

用星号表示未知或任何词："a * saved is a * earned"

快速预览：Google 搜索

使用OR进行多个选项搜索#

搜索多个可能的词：keyword1 OR keyword2

快速预览：Google 搜索

使用 site: 进行站内搜索#

限定在特定网站内搜索：site:example.com keyword

快速预览：Google 搜索

使用 filetype: 搜索特定文件类型#

搜索特定类型的文件：filetype:pdf keyword

快速预览：Google 搜索

高级技巧#

使用 inurl: 搜索URL中的特定词#

搜索URL包含特定词的页面：inurl:keyword

快速预览：Google 搜索

使用 intitle: 搜索标题中的特定词#

搜索标题中包含特定词的页面：intitle:keyword

快速预览：Google 搜索

使用 define: 查询定义#

查询词语的定义：define:keyword

快速预览：Google 搜索

使用 related: 查找相关网站#

查找与特定网站类似的网站：related:example.com

快速预览：Google 搜索

使用 cache: 查看网页的缓存版本#

查看Google的缓存页面：cache:example.com

预览：Google 搜索输入：cache:wikipedia.org

使用 AROUND(N) 进行近似匹配#

查找两词之间间隔不超过N个词的结果：keyword1 AROUND(N) keyword2

快速预览：Google 搜索

使用 allintext: 搜索正文中的所有词#

查找正文中包含所有指定词的页面：allintext:keyword1 keyword2

快速预览：Google 搜索

使用 allintitle: 搜索标题中的所有词#

查找标题中包含所有指定词的页面：allintitle:keyword1 keyword2

快速预览：Google 搜索

特定信息查询#

天气#

查询某地天气：weather location

快速预览：Google 搜索

时间#

查询某地时间：time location

快速预览：Google 搜索

股票#

查询股票行情：stock symbol

快速预览：Google 搜索

计算#

进行数学计算：直接输入公式

快速预览：Google 搜索

也可以通过输入 calculator 或 calc 打开计算器。

单位转换#

进行单位转换：quantity unit to unit

快速预览：Google 搜索

定义词语#

查找词语定义：define:word

快速预览：Google 搜索

定时器/秒表#

输入 timer 或 stopwatch 即可打开定时器/秒表。

综合示例#

假设你需要查找与AI相关的最新论文，但不希望看到广告或与招聘相关的内容，并且只想查找PDF文件。你可以使用以下组合搜索：

text
1

"artificial intelligence" -advertisement -jobs filetype:pdf

快速预览：Google 搜索

使用这些技巧，可以显著提高在 Google 上的搜索效率，快速找到更精准的信息。
