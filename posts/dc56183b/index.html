本文基于 go version go1.21.6 windows/amd64

GOPATH#

Windows 下，GOPATH 默认在 %USERPROFILE%\go 目录下。可通过编辑环境变量进行修改（以 D:\go 为例）：

变量名值
GOPATHD:\go

镜像服务#

goproxy/goproxy.cn

Go 1.13 及以上（推荐）#

打开你的终端并执行

bash
1
2

go env -w GO111MODULE=on
go env -w GOPROXY=https://goproxy.cn,direct

完成。

macOS 或 Linux#

打开你的终端并执行

bash
1
2

export GO111MODULE=on
export GOPROXY=https://goproxy.cn

或者

bash
1
2
3

echo "export GO111MODULE=on" >> ~/.profile
echo "export GOPROXY=https://goproxy.cn" >> ~/.profile
source ~/.profile

完成。

Windows#

打开你的 PowerShell 并执行

powershell
1
2

$env:GO111MODULE = "on"
$env:GOPROXY = "https://goproxy.cn"

或者

新增系统环境变量：

变量名值
GO111MODULEon
GOPROXYhttps://goproxy.cn

完成。
